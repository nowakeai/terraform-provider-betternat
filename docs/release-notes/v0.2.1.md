# v0.2.1

## Runtime Artifact Manifest Update

This provider patch adds built-in artifact manifest support for BetterNAT
runtime `v0.2.1`.

Use this release when GCP installs should use:

```hcl
betternat_version = "v0.2.1"
```

without explicit `agent_binary_url`, `agent_binary_sha256`, `cli_binary_url`,
or `cli_binary_sha256` overrides.

## What Changed

- Bumped the provider dependency on `github.com/nowakeai/betternat` to the
  `main` commit that includes the published `v0.2.1` runtime artifact
  checksums.
- Updated Registry docs and examples to use runtime `v0.2.1` and provider
  version `>= 0.2.1`.

## Compatibility

This is a provider patch release. It does not intentionally change Terraform
resource schemas, state shape, or replacement behavior.

## Validation

Release-candidate validation before tagging:

- `go test ./...`
- `go build ./cmd/terraform-provider-betternat`
- Terraform local dev-override validation for provider examples
- `betternat_runtime_artifacts` smoke for `v0.2.1` on `linux/amd64`

The GitHub Release workflow builds registry-compatible provider archives,
checks `SHA256SUMS`, signs the checksums, and uploads release assets.

## Artifact Integrity

Verify downloads with the attached provider `SHA256SUMS` file and signature.
