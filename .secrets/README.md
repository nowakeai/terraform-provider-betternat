# Local Secrets

This directory is for local release-signing material used during provider
release tests.

Do not commit private keys, passphrases, tokens, or downloaded secret files.

Expected local filenames:

```text
gpg-private-key.asc
gpg-passphrase.txt
```

The canonical copy is stored in 1Password:

```text
BetterNAT Terraform Registry GPG Signing Key
```
